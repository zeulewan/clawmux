# Roadmap

## What's New (v0.5.0)

ClawMux is in active development. Here's what shipped recently and what's coming next.

### Shipped

Session adoption on hub restart, so agents survive reboots. Fixed voice overlaps, first-word clipping, tab-switch audio dropouts, and missing typed messages. Added interjection support so you can talk while the agent is working. Sidebar navigation with status dots and unread indicators. Karaoke-style word highlighting during playback. Keyboard shortcuts for recording and session switching. Pause and interrupt controls. Multi-client sync across browser tabs. Renamed to ClawMux.

### Coming Soon

Streaming TTS for faster time-to-first-audio. Code block rendering in agent responses. Mobile improvements with better touch targets and wake lock. Live reload so the browser updates when the UI changes. ClawMux CLI for inter-agent messaging.

### iOS

Live Activity improvements for Dynamic Island. Editable STT transcription before sending. Action Button integration. Background mode refinements. Liquid Glass design for iOS 26.

## Future (v0.6.0)

Dual backend support so the hub can run agents through OpenClaw in addition to Claude Code. Sub-agent workers that let each agent spawn helpers. Real-time status visibility showing what agents are doing between voice calls. One-command setup script.
