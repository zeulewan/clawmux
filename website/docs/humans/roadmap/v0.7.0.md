# v0.7.0 - Agent Orchestration & Workspace

Sub-agent workers, workspace organization, and agent personalities.

## Workspace Management

- [ ] Drag-and-drop agents — rearrange agent cards in the sidebar by dragging.
- [ ] Project folders — organize agents into folders and subfolders by project.
- [ ] Create agents from browser — spawn new agent sessions directly from the web UI.

## Agent Personalities

- [ ] Distinct personalities — each voice gets a subtle personality to increase memorability (e.g., Sky is direct, Alloy is thoughtful, Echo is playful).

## Sub-Agent Workers

- [ ] Agents can spawn lightweight workers that inherit their voice and appear nested in the sidebar.
- [ ] Single-level hierarchy — workers can't spawn their own workers.
- [ ] Auto-terminate after 30 minutes idle, max 4 workers per leader.

## iOS

- [ ] Live Activity revamp — full Dynamic Island with waveform in auto mode, compact in PTT mode.
- [ ] STT edit before send — editable transcription after PTT recording.
- [ ] Action Button integration — assign "Toggle Recording" to iPhone Action Button.
- [ ] Background mode improvements — proper audio session deactivation, background VAD tuning.
- [ ] Liquid Glass design — adopt Apple's Liquid Glass visual language for iOS 26.

## Other

- [ ] Mobile improvements — optimized touch targets, haptic feedback, wake lock.
- [ ] Public API — token auth, versioned endpoints, agent management.
- [ ] Whisper model selection — choose model size from settings.
- [ ] Demo video — record and embed on the homepage.
- [ ] Troubleshooting guide — audio quality, latency, Tailscale, common failures.
